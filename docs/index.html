About

Aerial light, instant control

Helios is an open-source mobile app that connects directly to a companion computer running on your drone — a Raspberry Pi paired with a 50W LED spotlight module capable of 4,500–6,000 lumens of output.

Point your phone's camera at any location, tap capture, and your GPS coordinates are sent directly to the drone over WiFi. Within half a second, the LED activates and illuminates the target area.

The entire pipeline — from targeting to illumination — runs on hardware you own. No cloud service, no subscription, no approval queue. You own the drone, you control the light.

Illuminate active construction sites for night-shift crews without installing permanent lighting infrastructure. Reposition the beam instantly as work shifts to different areas of the site.

🚨

Direct a high-lumen spotlight to search areas during night operations with GPS-precise targeting. Coordinate aerial illumination with ground teams via the mobile app in real time.

Agriculture

Supplement lighting for greenhouses, night harvests, or field inspections. A drone-mounted LED covers large areas quickly with no permanent electrical infrastructure required.

🚒

Emergency Response

After natural disasters knock out power grids, a drone provides instant illumination for first responders — no generators, no fuel logistics, just a battery and a bright LED.

🔬

Field Research

Deliver controlled, on-demand illumination for night-time ecological surveys, archaeological excavations, or geological inspections in remote locations without power access.

Why Helios

Own your light source

Commercial drone lighting services charge per-flight fees, require pre-approval, and lock you into proprietary hardware. Helios runs on a $500 custom build you own outright.

Cost

One-time hardware cost.

A complete illumination-capable drone rig costs under $500 to build — Raspberry Pi, 50W LED module, MOSFET driver, and a flight controller. No monthly fees, no per-minute billing, no enterprise tier. Helios is free and open source.

Control

Direct WiFi connection

Your phone connects directly to the drone's WiFi access point — no cloud relay, no latency introduced by a remote server. Sub-500ms from command tap to LED activation. Full WebSocket and HTTP protocol documentation built into the app.

Speed

Point, tap, illuminated

Other platforms require web dashboards, coordinate entry forms, and flight plan submissions. Helios uses your phone's camera and GPS — aim at a location, tap capture, and the LED activates in under a second. The entire flow takes under 5 seconds.

Freedom

Works on any drone platform

The drone-server runs on any Raspberry Pi connected to a relay or MOSFET-driven LED. Compatible with custom ArduPilot builds, converted commercial drones, or tethered platforms. No vendor lock-in, no DJI SDK required.

Get Started

Up and running in minutes

No accounts. No API keys. Clone the repo, install dependencies, and connect to your drone.

1

Clone & install the app

Clone the repository and install Expo dependencies.

git clone https://github.com/joedhawan/repo-local.git cd repo-local npm install --legacy-peer-deps npx expo start
2

Start the drone server

On your Raspberry Pi (or any machine for mock testing), start the companion server.

cd drone-server npm install # Mock mode (no GPIO hardware needed) npm run dev:mock # Production on RPi with real LED NODE_ENV=production npm start
3

Connect your phone to the drone WiFi

Connect to the HELIOS-DRONE WiFi network broadcast by the Raspberry Pi access point.

4

Configure & connect in the app

Open the Settings tab, enter 192.168.4.1 as host and 8080 as port, then tap Connect to Drone. Use the Camera tab to illuminate any GPS target. Set host to demo to simulate without hardware.
